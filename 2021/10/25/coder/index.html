CODER

自创 CODER 学习法，每天坚持完成五类任务

  • C - Coding
  • O - Open Language
  • D - Dedao App
  • E - Exercise
  • R - Reading

Coding

坚持每日写代码，保持逻辑性思维。可以是工作上的编码，也可以是开源项目的实践，或者是刷一道 LeetCode 算法题。

Open Language

英语的持续学习是有必要的，不管是生活还是工作。OpenLanguage (开言英语) 是本人这些年尝试过的学习英语的最适合自己的 App (也许和自己喜欢听 Podcast 有一定关系)，App 里的激励、社交机制都做得很不错（组队打卡、瓜分奖金、奖学金），能给人一种确定性 “我每天都至少能学习十分钟英语”，坚持下去，一定会有一天量变引起质变的。

Dedao App（得到）

得到 App 是一个能给我源源不断提供广度知识的产品，从个人的成长、职业发展到法律、金融甚至生命科学都有，里面的知识都很接地气。得到 App 上的课程能提供一个很好的入门基础知识，也能给你一个清晰的学习路径，可以在早期少走弯路。当然，如果要深入学习某个知识，仅使用得到那是远远不够的。（By the way，我不觉得现在的得到是在贩卖焦虑）

Exercise 运动

生命在于运动。如果有什么能导致最后写不了代码的，也许就是脖子（颈椎病）了。预防各种职业病最好的方式就是保持运动习惯。跑步、平板支撑、深蹲、划船机、甚至滑板、滑雪，做自己喜欢的运动，让身体持续保持最佳状态。

Reading 阅读

得到 App 能解决了知识广度的问题，但知识的深度还是得靠有选择性地去阅读，不管专业性或者非专业性的，找到知识的源头，静下心来去阅读它，然后实践它。这里的阅读可以是看一篇技术 blog、一篇论文，或者是 GitHub 上的一份源代码、一个 issue solution，也可以是一本书、一节极客时间或者得到上的课程。

#加油！坚持终身学习！
