How To Choose Spring Cloud Version

最近在搭 Spring Cloud 环境时，因为 Spring Boot、Spring Cloud 及相关组件版本的问题，常报错。于是花时间 google 了下相关版本的知识，总结如下。

Spring 版本说明

官网标识的版本主要有 GA、SNAPSHOT、PRE 三个。

Spring Boot 版本选择

查看最新版本 https://spring.io/projects/spring-boot#learn

当前官方推荐使用的 GA 版本是 2.6.0。

点击 Reference Doc. 可以查看参考文档。在参考文档页面下点击 Dependency Versions 可以查看相关依赖组件版本说明。如 SpringBoot 2.6.0 下声明使用 Netty 依赖时，若不指定版本，则默认使用 Netty 4.1.70.Final 版本。

Spring Cloud 版本选择

查看最新版本 https://spring.io/projects/spring-cloud#learn

当前官方推荐使用的 GA 版本是 2020.0.4。（Spring Cloud 从 2020 开始不再使用伦敦地铁站命名版本号，所以 Hoxton 是最后使用地铁站命名的版本号了）

点击 Reference Doc. 可以查看参考文档。（可以看到 Version: 2020.0.3，Supported Boot Version: 2.4.6）

如果需要在已有 Spring Boot 应用中加入 Spring Cloud 的话，可以参考 adding-spring-cloud-to-an-existing-spring-boot-application 表格。

结论

综上，当前 Spring Boot 3.5.7 GA 版本推荐使用 Spring Cloud 2020.0.x (2020.0.4) 版本。
