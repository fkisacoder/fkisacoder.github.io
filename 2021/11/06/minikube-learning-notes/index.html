Minikube Learning Notes

0. background info

  • 操作时使用系统 macOS Big Sur
  • 已安装 Docker Desktop version 3.4.0
  • 已安装 kubectl

1. Mac 下安装 minikube

使用阿里云构建的版本避免网络超时报错问题。 : (

  • 安装
  • 启动 minikube
  • 启动 Dashborad 查看下可视化界面（Ctrl + C 关闭）

2. 确认 kubectl 已正确安装

3. deploy docker image

  • 切换至 minukube 下的 docker-deamon

进入 SpringBootDocker 项目根目录（该项目可查看 A demo of SpringBootDocker），执行 docker 环境切换命令。

minikube docker-env 会输出什么？可以 echo 看下。

  • build docker image

在此之前已经执行过 mvn package 生成了 jar 文件。若 target 目录下还没生成 jar 文件，需先执行 mvn package。

  • 或者，切换至 minikube 的 docker daemon 后，直接运行 mvn 命令打包 docker image

4. create deployment

  • 创建 springboo-demo-deployment.yaml 文件

Kubenetes 1.9 之前的版本 apiVersion 需为 v1beta1

imagePullPolicy 设置为 IfNotPresent (本地没有才去 pull) 或 Never

  • 创建 deployment
  • 创建服务
  • 访问服务
  • 新开一个 terminal 访问应用。

It works.
